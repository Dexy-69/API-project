طيب هنا بذا القسم بعلمك كيف تحمل المكاتب المطلوب طبعا بعد ما تسوي بيئة افتراضية للبايثون تلقا شرحها في ملف (README-env.txt)

طيب الان عشان تحمل المكاتب المطلوبة منك عشان تقدر تشغل المشروع تحتاج تكتب ذا الامر

(pip install -r requirements.txt)

اكتب الامر كامل الي بين الاقواس الي فوق و وقتها بتتحمل كامل المكاتب المطلوبة لتشغيل المشروع (السيرفر)
